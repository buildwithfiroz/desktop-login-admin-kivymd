> [!IMPORTANT]
>
> ### Why I Built This
> While working with a client, I faced a common problem ‚Äî they wanted a simple desktop login system that could connect to their CRM. The idea was straightforward:
>
> - A user logs in from a desktop app  
> - The admin gets notified in the CRM  
> - Admins can control who logs in, edit credentials, or add/remove users  
> - Both users and admins access the **same app**, but are shown different interfaces based on their access level
>
> But most solutions were either too complex or web-only.
>
> So I built this prototype ‚Äî a clean, functional **desktop app using Kivy and KivyMD**, featuring a login system and an admin panel that mimics how this workflow would operate in production.
>
> It‚Äôs simple, visual, and built to show exactly how such a system can integrate with CRM logic ‚Äî while keeping the experience smooth for both users and admins from a single access point.

> ### Getting Started is Super Easy 
>
> This project isn‚Äôt just easy to use ‚Äî it‚Äôs also **easy to set up and integrate into your own system**. Follow the steps below to get it running on your machine in minutes:
>
> **Step 1:**  
> Clone the repository  
> ```bash
> git clone https://github.com/buildwithfiroz/desktop-login-admin-kivymd.git
> ```
>
> **Step 2:**  
> Move into the project folder  
> ```bash
> cd desktop-login-admin-kivymd
> ```
>
> **Step 3:**  
> Create a virtual environment  
> ```bash
> python3 -m venv myenv
> ```
>
> **Step 4:**  
> Activate the virtual environment  
> - On macOS/Linux:  
>   ```bash
>   source myenv/bin/activate
>   ```
> - On Windows (PowerShell):  
>   ```powershell
>   .\myenv\Scripts\Activate.ps1
>   ```
>
> **Step 5:**  
> Install the dependencies  
> ```bash
> pip install -r requirements.txt
> ```
>
> **Step 6:**  
> Run the app to view admin pannel 
> ```bash
> python admin.py
> ```
> <br>
>
> Run the below cmd to view the login window
> ```bash
> python login.py
> ```
> 
> That‚Äôs it! You‚Äôre now up and running. Feel free to customize or extend the app to fit your workflow.
